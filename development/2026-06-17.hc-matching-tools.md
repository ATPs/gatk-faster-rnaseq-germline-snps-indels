# HaplotypeCaller matching tool suite

## Purpose

This records the first implementation slice for making the Rust HaplotypeCaller match Java GATK stage by stage. These tools do not change the pipeline default. They create the comparison and oracle harness needed before implementing the next GATK-like caller internals.

## Added Rust binaries

- `rust_hc_vcf_compare`: exact allele-key VCF comparison with Markdown, TSV, JSON, and detail outputs.
- `rust_hc_select_regions`: focused debug-region selector from GATK-private, Rust-private, shared genotype-different, shared indel, and shared-match calls.
- `rust_hc_run_gatk_oracle`: reproducible GATK HaplotypeCaller oracle command runner using existing GATK debug outputs.
- `rust_hc_active_region_diff`: TSV/flat-JSONL stage diff for active-region records.
- `rust_hc_read_finalize_diff`: TSV/flat-JSONL stage diff for finalized-read records.
- `rust_hc_assembly_diff`: TSV/flat-JSONL stage diff for assembly, haplotype, and event records.
- `rust_hc_pairhmm_diff`: TSV/flat-JSONL stage diff for PairHMM likelihood records, default tolerance `1e-3`.
- `rust_hc_genotype_diff`: TSV/flat-JSONL stage diff for genotype and annotation records.
- `rust_hc_acceptance_report`: Markdown aggregator for VCF and stage summaries.

Shared implementation is in `src/rust/src/hc_tools.rs`.
